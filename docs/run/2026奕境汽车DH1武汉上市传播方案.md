## 4. Research-Based Insights

### 4.1 核心发现

**发现一：问界M9统治50万+市场，但"华为光环"同质化风险增大**

问界M9大改款（2026年3月工信部公示）车身加长至5285mm、增程纯电续航335km、896线激光雷达上车，月销最高1.54万辆，连续21个月50万+销冠。45%车主为BBA置换用户，智能化体验是核心转化驱动。

→ **对奕境的启示**：奕境同为华为乾崑"境系列"嫡系，技术栈与问界高度同源。若传播策略停留在"又一个华为系"，将陷入同质化困局。必须找到问界M9不具备的差异化叙事——**家庭场景深度+东风安全底蕴+灵动角窗独创卖点**。

**发现二：理想L9销量下滑但"移动的家"心智占据领先**

理想L9前5月销量同比下滑40%，但"冰箱彩电大沙发"的家庭生态体验深入人心。L9 Livis定价55.98万，800V全主动悬架+5C超充，是DH1最直接竞品。

→ **对奕境的启示**：理想已占据"移动的家"心智，奕境不宜正面争夺"家庭"标签，而应切入更精准的"六座真平权"——特别是**第三排尊严**，这是理想L9口碑中被诟病最多的点（后备箱空间小、第三排坐垫厚度不足）。

**发现三："灵动角窗"是DH1独创卖点，竞品无同类配置**

DH1三排侧窗采用灵动玻璃角窗，支持自定义画面、触控和交互功能，专为儿童/家庭第三排体验设计。问界M9无此配置，理想L9仅有后排娱乐屏。

→ **对奕境的启示**：灵动角窗=差异化利器，可成为传播引爆点——将"第三排"从"被忽视的角落"升级为"全家最想坐的位置"。

**发现四：武汉"车谷"地域认同是独特传播资产**

武汉经开区新能源车产量占比超全国平均3.4个百分点，"车谷造"已成武汉城市名片。奕境背靠东风总部武汉，有"本土品牌回家"的地缘认同优势。

→ **对奕境的启示**：问界M9（赛力斯/重庆）、理想L9（北京/常州）在武汉无此地域根基。东风57年武汉根基=信任加速器。

## 5. Strategic Direction

### 战略定位

**核心策略**：以"灵动角窗"为创意原点，以"第三排尊严"为传播切入，以武汉长江夜景为体验场景，以东风57年本土根基为信任锚点，构建"角窗实验室·境归江城"武汉上市传播IP。

**策略逻辑**：

```
品牌信任真空期（新品牌首战）
        ↓ 解决方案
"造境者说"深度内容（工程师故事→技术信任）
        ↓
"不卖车先卖第三排"（反常规悬念→好奇心引爆）
        ↓
"角窗实验室"线下体验（角窗互动→家庭情感共鸣）
        ↓
"境见长江"夜试发布会（场景化体验→到店转化）
        ↓
"车谷主场上新"（政企资源叠加→后效提升）
```

**与奕境品牌调性的契合**：
- 家庭化：角窗实验室="每个座位都有尊严"，契合奕境偏家庭化品牌调性
- 科技感：华为乾崑全栈技术="满血华为"，但通过角窗体验转化为"可感知的科技"
- 安全感：东风57年造车底蕴+零缺陷品控，是问界/理想不具备的信任资产
- 武汉主场：东风=武汉，"境归江城"是其他竞品无法复刻的地域叙事

## 7. Core Idea Options

### 方向一（Primary）：角窗实验室·境归江城

**创意核心**：以"灵动角窗"为核心创意原点，武汉长江夜景为体验场景，打造奕境DH1武汉上市传播IP。

**三段式结构**：
- 前期："不卖车先卖第三排"悬念传播+"我家第三排"UGC启动+"造境者说"播客预热
- 中期："境见长江"夜试发布会+角窗实验室快闪+KOL体验分发
- 后期：UGC成果展+车谷开放日+后效数据复盘

**创新性说明**：
1. **反常规上市策略**：打破"新车上市=全车卖点发布"的行业惯例，聚焦"第三排"单点突破，制造认知冲突和好奇心
2. **独创产品力转化为体验IP**：灵动角窗=竞品无同类配置，从产品功能升级为可参与的线下体验空间
3. **武汉独有场景嫁接**：长江夜景试驾=地理不可复制，城市自豪感驱动社交传播

### 方向二（Co-direction）："造境者说"视频播客

**创意核心**：东风老匠人与华为新锐工程师的深度对谈，用"人"的故事建立技术信任。

**创新性说明**：
1. **新格式红利**：B站视频播客消费时长暴涨270%（2025Q1），汽车行业尚无先例
2. **双品牌故事解构**：57年制造底蕴 vs 896线激光雷达，反差叙事天然吸睛
3. **深度内容=信任杠杆**：长内容构建技术可信度，短视频切片扩大触达

### 方向三（Co-direction）："车谷主场上新"

**创意核心**：东风品牌回归武汉主场的情感叙事+政企资源叠加转化。

**创新性说明**：
1. **地域情感不可替代**：东风57年武汉根基，问界/理想无此资产
2. **政企资源效率最大化**：武汉经开区汽车消费券+购车补贴+产业集群参观，匹配招标"区域特点+热点事件+重点资源"要求
3. **后效提升闭环**：车谷开放日→品鉴→政策叠加→持续引流
