## 1. Mission & Key Questions

以"奕家主场 · 境享美好"为核心主张，在武汉这座"中国车谷"主场，为奕境X9打造一场兼具央企底蕴与华为智能科技感的家庭旗舰SUV上市战役。目标是在6月上市窗口期内，建立奕境X9作为"50万级最懂中国家庭的6座智能SUV"的心智定位，实现声量引爆、圈层集客与高效转化的三重目标。

### 2.2 目标受众画像

**核心人群：武汉新中产家庭（30-45岁）**
- 家庭结构：二孩或计划二孩，三代同堂需求上升
- 收入水平：家庭年收入50-100万，可负担50万级车型
- 购车动机：空间刚需（6座）+ 智能体验（华为生态用户）+ 安全焦虑（家庭责任）
- 信息获取：抖音（汽车评测+生活方式）、小红书（宝妈圈层）、微信（私域决策）
- 决策特征：女性参与度高（影响最终决策60%+）、重视真实车主口碑、对"华为"有天然信任

**辐射人群：科技尝鲜者（25-35岁）**
- 华为生态深度用户（手机+平板+手表）
- 对智能驾驶有强烈好奇心，愿意为科技溢价买单
- 社交活跃，UGC内容生产意愿强

### 2.3 市场机会洞察

1. **"华为满血智能"空白区**：问界M9虽占华为先发优势，但消费者对其"完整华为技术栈"认知模糊。奕境X9可抢占"ADS 5.0+HarmonyOS 6.0+896线激光雷达+XMCD数字底盘"的"满血华为"定位。
2. **家庭6座SUV智能化洼地**：理想L9在家庭场景做到极致，但智能化（尤其智能驾驶）是其明显短板。奕境X9可打造"比理想更智能，比问界更懂家庭"的夹缝定位。
3. **武汉主场情感优势**：作为东风发源地、中国车谷核心城市，武汉消费者对"本土高端品牌"有天然情感倾斜，这是全国其他市场不具备的地缘红利。

## 3. Core Marketing Strategy

### 3.1 Core value（核心价值主张）

**央企造车底蕴 | 华为满血智能 | 家庭旗舰空间 | 极致安全守护**

- **央企造车底蕴**：东风57年造车基因，军工品质标准，不是新势力能复制的积淀
- **华为满血智能**：ADS 5.0高阶智驾 + HarmonyOS 6.0智能座舱 + 896线激光雷达 + XMCD数字底盘——"华为技术最全量产的SUV"
- **家庭旗舰空间**：5.3m车长+3.12m轴距，真6座全尺寸，第三排不再是"应急座位"
- **极致安全守护**：CAS 5.0安全体系，央企责任背书，给家人最高的安全承诺

### 3.2 Marketing主线（营销叙事主线）

## **奕家主场 · 境享美好**

叙事逻辑：
- **"奕家"** = 奕境汽车 × 每一个中国家庭，品牌成为家庭的一份子
- **"主场"** = 武汉是东风主场、中国车谷主场、更是每个家庭的生活主场
- **"境享美好"** = 在奕境X9的空间里，尽享科技、安全、舒适带来的美好生活

### 3.3 Strategy pillars（策略支柱）

| 支柱 | 定位 | 核心任务 | 对应执行模块 |
|------|------|---------|------------|
| **声量饱和攻击** | 破圈认知 | 3个月内让武汉目标人群100%知晓奕境X9 | 英雄事件+城市地标+短视频矩阵 |
| **场景高频体验** | 建立信任 | 让1000+家庭真实触摸、试驾、体验产品 | 家庭体验计划+快闪店+试驾活动 |
| **链路高效转化** | 收割订单 | 从兴趣到留资到试驾到订单的最短路径 | 私域运营+门店承接+促销政策 |
| **口碑裂变引擎** | 持续发酵 | 将首批车主转化为品牌传教士 | KOC内容+车主社群+转介绍激励 |

### 3.4 Core Strategic Idea

**One-line claim（一句话主张）**：
> **"中国车谷出品，满血华为智能——给家人的旗舰，从不将就。"**

**Core contradiction（核心矛盾）**：
消费者默认"新势力=智能，央企=保守"，不相信央企能造出真正智能的旗舰SUV。

**Resolution path（破局路径）**：
以"华为满血技术栈"的可视化体验（ADS 5.0城市NOA现场演示、HarmonyOS 6.0多设备流转、896线激光雷达实时点云展示）直接颠覆"央企不智能"的刻板认知；以"武汉主场"地缘情感+"家庭场景"真实代入，建立品牌温度。

## 11. Defensibility（防御性）

**为什么竞争对手无法简单复制**：

1. **地缘独占性**："武汉主场+中国车谷"的身份是奕境独有的地缘资产。问界（重庆）、理想（北京）、蔚来（合肥）都无法复制"在武汉发布=在车谷发布"的情感逻辑。

2. **技术完整性**："华为满血技术栈"（ADS 5.0+HarmonyOS 6.0+896线激光雷达+XMCD数字底盘）是行业最全的华为技术搭载，竞争对手即使单点超越，也无法在"完整性"上匹敌。

3. **央企信任壁垒**：东风57年造车历史+军工品质背书，是新势力无法在短期内建立的信任资产。在"安全"这个家庭购车核心决策因素上，央企身份是天然优势。

4. **体验家庭种子网络**：100组真实家庭的深度体验和口碑传播，形成了竞争对手无法快速复制的"真实用户护城河"。

5. **文化和自然遗产日主场身份**：2026年武汉文化和自然遗产日主会场活动的官方背书，是花钱也买不到的稀缺资源。
