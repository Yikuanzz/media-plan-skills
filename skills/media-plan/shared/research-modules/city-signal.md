# Research module: city and regional signal

**When to load:** Regional launch, LBS events, or **Hotspot × Regional Resource** blocks.

**Before** web search, read this file and [../query-playbook.md](../query-playbook.md) regional items.

## Query focus

- City name + `消费` / `商圈` / `活动` + season / month from brief
- `地标` + category + `打卡`
- Local sports / fairs / large public events in the plan window (cross-check dates)

## Output

- Support `## Hotspot × Regional Resource Match` and `Local life map` with **named** places, venues, or calendar hooks.
